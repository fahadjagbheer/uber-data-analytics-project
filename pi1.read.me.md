# Uber Data Analytics Dashboard — Power BI

An interactive business intelligence dashboard built with **Microsoft Power BI**, analyzing Uber ride data across bookings, revenue, vehicle types, rider behavior, and locations.

---

## Project Overview

This project analyzes Uber ride data to uncover business insights around booking performance, revenue trends, rider ratings, and vehicle contribution. The dashboard is designed for business decision makers to monitor KPIs and explore patterns interactively.

---

## Dashboard Structure

The report consists of **2 pages**, designed to provide a clear and focused analysis.

### Page 1 — Introduction

This page provides an overview of the company, including:

- Company vision and strategic objectives
- General company background and business context

### Page 2 — Analysis & Visualizations

This page contains all detailed insights and interactive visuals, including:

- Key Performance Indicators (KPIs): total rides, completed rides, incomplete rides, revenue, and average distance
- Charts showing trends over time (monthly/quarterly)
- Revenue breakdown by vehicle type
- Rider behavior and cancellation analysis
- Location-based insights and demand patterns
- Interactive filters for deeper exploration
